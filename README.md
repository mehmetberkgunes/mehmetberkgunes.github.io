# Mehmet Berk Güneş - GitHub Pages Academic Website

Bu paket, GitHub Pages üzerinde doğrudan yayımlanabilecek temiz bir akademik CV sitesidir.

## İçerik

- `index.html` — ana web sayfası
- `style.css` — tasarım dosyası
- `assets/profile.jpg` — profil fotoğrafı
- `assets/Mehmet_Berk_Gunes_CV.pdf` — indirilebilir CV
- `.nojekyll` — GitHub Pages için sade statik site ayarı

## GitHub'a yükleme

1. `mehmetberkgunes.github.io` repository'sine girin.
2. Eski `index.html`, `style.css` ve `assets` klasörünü silin veya bu dosyalarla değiştirin.
3. Bu ZIP içindeki dosyaları çıkarın.
4. ZIP'in içindeki dosyaları repository'nin ana dizinine yükleyin:
   - `index.html`
   - `style.css`
   - `.nojekyll`
   - `assets/`
5. En altta `Commit changes` butonuna basın.
6. GitHub Pages ayarı şu şekilde olmalı:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)

## Site adresi

https://mehmetberkgunes.github.io

## CV güncelleme

Yeni CV PDF dosyasını her zaman şu isimle yükleyin:

`assets/Mehmet_Berk_Gunes_CV.pdf`

Bu isim korunursa sitedeki Download CV butonu bozulmaz.

## Fotoğraf güncelleme

Yeni fotoğrafı şu isimle yükleyin:

`assets/profile.jpg`

Bu isim korunursa sitedeki fotoğraf otomatik güncellenir.

## Gizlilik notu

Telefon numarası bu public sitede gösterilmemiştir. İletişim alanında yalnızca e-posta, ORCID ve YÖKSİS Researcher ID yer almaktadır.
